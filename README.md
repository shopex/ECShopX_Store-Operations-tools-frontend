# Getting started
Node.js (current LTS) and npm are required to run the project. To be sure about the version compatibility you can enable Node's corepack.

### Node
```
Node v12.20.1
```

### Installation
```
cd ecshopx-dianwu
npm i
```

### Configure the .env file
```shell
# Backend API Base URL
APP_BASE_URL=
```

### Run project 
```
npm run dev:h5
```

### Build packages 
```
npm run build:h5
```
